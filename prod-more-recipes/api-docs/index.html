    Introduction

    More Recipes API Documentation

    More-Recipes provides a platform for users to share the recipe ideas they have invented or learnt. This gives them the opportunty to get reviews from other users all around the world.

    This API documentation page was created with Slate.

    Base Url

    These are the base url which are to prepend request routes for API consumption

    Heroku : https://prod-more-recipes.herokuapp.com/api/v1/

    Local : http://localhost:8081/api/v1/

    field Description

    This returns an object containing updated recipe details and recipe image is uploaded to cloudinary.
